Tools

Arduino (similar to C), RFID tags, 3d printing

Discription

This is a school project for a real client. The client is an escape room builder. The project was to invent and realize a escape room puzzle that fitted within a cyber security escape room.

A puzzle was realized where the goal was to scan a code in the correct order. This was done with RFID tags. The puzzle itself also contains the ability to reprogram a new code onto the RFID tags.
